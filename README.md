# Практична робота "Масиви, вирази, керування виконанням програми"

Цей репозиторій містить стартовий код для виконання практичної роботи, який містить заготовки двох класів, що демонструють структуру проекту для виконання завдань на масиви, керуючі конструкції та вирази.

## Опис завдання
Для виконання роботи було обрано варіант №15, що передбачає розробку алгоритму для знаходження другого за величиною числа у заданому масиві.

## Технічні особливості
Реалізація забезпечує лінійну швидкість обчислення шляхом одноразового проходу по масиву, що мінімізує навантаження на процесор. Використано оптимальні примітивні типи даних для мінімізації обсягу використовуваної пам'яті. Логіка програми враховує можливість наявності дублікатів та коректно визначає наступне унікальне значення після максимуму. Код оформлено згідно з рекомендаціями щодо іменування змінних та структури об'єктно-орієнтованих програм.
